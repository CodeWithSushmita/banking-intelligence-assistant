Title

HDFC Banking Intelligence Assistant (Multi-Agent GenAI System)


Overview

A multi-agent AI system that intelligently routes banking queries between:

- Policy Q&A (RAG)
- Data queries (SQL)

Architecture

- LangGraph orchestrator
- RAG (FAISS + MiniLM)
- SQL Agent
- Streamlit UI

Features

- Intelligent routing
- Grounded responses
- Clickable source attribution
- Explanation layer ("Why this answer")

Tech Stack

- LangChain
- LangGraph
- FAISS
- HuggingFace Embeddings
- Groq LLM
- Streamlit
